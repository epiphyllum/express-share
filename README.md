#  API

提供/api/get_a服务 : 返回一个随机数{ a : a }  
提供/api/get_b服务 : 返回一个随机数{ b : b }  
提供/api/add服务   : 返回加法结果{ result: r }  

服务启动在3000端口

启动方式:

```
npm install;   // 先安装
babel-node app.js  // 启动
```


# demo
